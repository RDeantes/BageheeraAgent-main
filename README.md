# 🤖 Bagheera RH Automation System

An AI-powered Human Resources automation system that manages employee contracts, expiration tracking, and work anniversaries through a Telegram bot, using Excel as a dynamic data source.

---

## 🚀 Overview

Bagheera is a multi-agent system designed to automate repetitive HR processes such as:

- Contract generation
- Expiration tracking
- Vacation (work anniversary) planning

The system interacts with users through a Telegram bot and uses an Excel file as a lightweight database.
